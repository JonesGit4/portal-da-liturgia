## O Problema Central

O site é **React CRA puro — Client-Side Rendering**. Toda página retorna apenas:

```html
<div id="root"></div>
```

São +490 músicas, orações, artigos, roteiros — conteúdo valiosíssimo, mas **invisível** para Google, Bing, redes sociais e IA. O crawler vê 1.371 bytes e vai embora.

## Os 5 Problemas Mais Graves

1. **CSR puro** — sem SSR/SSG, Google pode até renderizar mas perde frescor, Bing/DuckDuckGo nem tentam
2. **Meta description padrão CRA** — "Web site created using create-react-app" + `lang="en"` em site português
3. **Robots.txt** não referencia sitemap, não libera bots de IA
4. **Sitemap congelado** desde 04/02/2025, 494 URLs, prioridades erradas
5. **Zero schema markup** — sem JSON-LD, sem rich snippets

---

## Recomendações (Prioridade)

1. Migrar de CRA para Next.js com SSG/SSR
2. Implementar next-seo + schema markup + sitemap dinâmico
3. Corrigir robots.txt + liberar bots IA + criar `/llms.txt`
4. Fase 2: Cache Cloudflare, Core Web Vitals
5. Fase 3: Monitoramento Search Console, SEO programático
